## How to use
1. Create a GitHub repository.
2. Upload all files and folders.
3. Add your images inside the `images` folder.
4. Replace the placeholder GitHub and LinkedIn links.
5. Upload your resume using the exact filename `Jeric_Albar_Resume.pdf`.
6. Open repository Settings > Pages.
7. Select "Deploy from a branch".
8. Choose the `main` branch and `/root`.
9. Save and wait for GitHub Pages to publish.
